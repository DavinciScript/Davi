Functions

Array

Append

append(list, value1, value2, ...)

Append values to a array.

Example

append([1, 2], 3, 4) => [1, 2, 3, 4]

// output: [1, 2, 3, 4]

Range

range(n)

Generate a list of integers from 0 to n-1.

Example

range(3) => [0, 1, 2]

// output: [0, 1, 2]

Slice

slice(str or list, start, end)

Get a substring or sublist from a string or list.

Example

slice("hello", 1, 3) => "el"

// output: "el"

Sort

sort(list, [key])

Sort a list of values.

Example

sort([3, 1, 2]) => [1, 2, 3]

// output: [1, 2, 3]

Conversion

Int

int(value)

Convert a value to an integer.

Example

int("42") => 42

// output: 42

Str

str(value)

Convert a value to a string.

Example

str([1, 2, 3]) => "[1, 2, 3]"

// output: "[1, 2, 3]"

File System

File Get Contents

fileGetContents(url)

Get the contents of a file or URL.

Example

fileGetContents("http

// output: "..."

HTTP

HTTP Register

httpRegister(pattern, handler)

Register a handler function for a URL pattern.

Example

httpRegister("/", func() { return "Hello, World!" })

// output: "Hello, World!"

HTTP Listen

httpListen(portOrAddress)

Start the HTTP server.

Example

httpListen("

// output: Server is starting on http

String

Length

len(value)

Get the length of a string, list, or map.

Example

len("hello") => 5

// output: 5

Char

char(string)

Convert an ASCII code to a character.

Example

char(65) => "A"

// output: "A"

Lower

lower(string)

Convert a string to lowercase.

Example

lower("HELLO") => "hello"

// output: "hello"

Find

find(haystack, needle)

Find the first occurrence of a substring in a string or a value in a list.

Example

find("hello", "e") => 1

// output: 1

Join

join(list, separator)

Join a list of strings into a single string with a separator.

Example

join(["a", "b", "c"], ", ") => "a, b, c"

// output: "a, b, c"

Rune

rune(str)

Convert a 1-character string to an ASCII code.

Example

rune("A") => 65

// output: 65

Split

split(string, [separator])

Split a string into a list of substrings.

Example

split("a, b, c", ", ") => ["a", "b", "c"]

// output: ["a", "b", "c"]

Type

type(value)

Get the type of a value as a string.

Example

type(42) => "int"

// output: "int"

Upper

upper(string)

Convert a string to uppercase.

Example

upper("hello") => "HELLO"

// output: "HELLO"

System

Exit

exit([code])

Exit the script with an optional exit code.

Example

exit(1)

// output: exit status 1

Read

read([filename])

Read the contents of a file or standard input.

Example

read("file.txt") => "contents of file.txt"

// output: "contents of file.txt"

Time

time(none)

Get the current date and time as a string.

Example

time() => "2018-01-01 12

// output: "2018-01-01 12

Args

args(none)

Get the command-line arguments passed to the script.

Example

args() => ["arg1", "arg2"]

// output: ["arg1", "arg2"]

Echo

echo(value1, value2, ...)

Print values to the standard output.

Example

echo("hello", 42) => hello 42

// output: hello 42
